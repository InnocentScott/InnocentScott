### Featured projects

**[EnChi](https://github.com/InnocentScott/enchi)** — English &amp; Chinese language-learning app
A cross-platform mobile client (React Native + Expo, TypeScript) backed by a polyglot microservices system: a NestJS content service, an Express spaced-repetition (SM-2) service, event-driven workflows over RabbitMQ, a database-per-service design on PostgreSQL + Redis, all behind a Traefik API gateway and orchestrated with Docker Compose.

**[Shopoo](https://github.com/shopoo-vn)** — electronics-marketplace microservices back-end
A 6-service backend with NestJS services for listings (full-text search) and admin, a real-time chat service (Express + Socket.io), RS256 JWT auth, per-service PostgreSQL, Redis, RabbitMQ and MinIO object storage — orchestrated with Docker Compose.
